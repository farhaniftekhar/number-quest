# Number Quest: The Rational Realm

An offline-first, Montessori-inspired math exploration for ages 9–12. Learners explore whole numbers, fractions, ratios, probability, and the feel of rational vs irrational numbers through calm visuals and manipulatives.

## Why it teaches well
- **Concrete → abstract**: Start with beads, then fade into fraction bars and number lines.
- **Control of error**: Visual fills and ratios show correctness without penalties.
- **Self-directed**: No timers. Learners choose when to advance.
- **Selective beads**: Beads model part–whole (fractions, ratios, probability). Irrationals avoid beads, highlighting their endless nature.

## Architecture
- **Vanilla TypeScript + Canvas**: No frameworks; ES modules compiled via `tsc`.
- **Core game loop**: `Game` handles update/render ticks, level switching, and progress text.
- **Pluggable worlds/levels**: Worlds live in `src/worlds`. Each level implements `Level` (`onEnter`, `update`, `render`, `isComplete`).
- **Reusable visuals** in `src/components`:
  - `numberLine` – ticks and highlights for rational/irrational contrasts.
  - `fractionBar` – shaded part–whole bars.
  - `beadBag` – drawable bead manipulative with history view.
  - `spinner` – weighted probability spinner.
  - `dice` – fair dice visualization.
- **UI shell**: `UIManager` wires DOM controls to level-provided actions.
- **Progress**: `LocalProgress` stores unlocked levels and last position in `localStorage`.
- **Offline-first**: `sw.js` caches HTML, CSS, and built JS for repeat visits.

## Folder structure
```
/index.html      – main page
/style.css       – calm UI styling
/sw.js           – service worker cache
/src
  /components    – reusable visuals (beads, bars, spinner, dice, number line)
  /core          – game loop, progress store, shared types
  /ui            – DOM control manager
  /worlds        – pluggable level implementations
  main.ts        – bootstraps UI, worlds, game
/dist            – compiled JS (generated by tsc)
```

## Implemented worlds & sample levels
1. **Bead Grove (Fractions)** – *Bead Grove: Compare Fractions*  
   Scoop beads into bars (3/4 vs 2/3). Learners feel part–whole using beads, then see bars.

2. **Chance Meadow (Probability)** – *Chance Meadow: Experimental Probability*  
   Repeated bead draws (6:3:1 ratios), spinner, and dice. Learners see experimental probability drift toward expected ratios.

3. **Rational Realm (Rational vs Irrational)** – *Rational Realm: Finite vs Endless*  
   Number line shows exact rationals vs endlessly stretching √2 and π. Repeating decimals (1/3) highlight why they remain rational.

## Extensibility
- Add a level: create a `Level` in `src/worlds`, add to `createWorlds` registry.
- Add a manipulative: place a component in `src/components` and render it in a level.
- Progress is automatic: mark completion in `isComplete`; `Game` unlocks and updates UI.

## Controls & feedback
- Buttons per level drive actions (scoops, spins, toggles).
- Feedback area narrates discoveries; no penalties.
- Navigation: Previous/Next level buttons; Reset progress clears local storage.

## Development
```bash
npm install
npm run build
# open index.html in a browser (dist/main.js is referenced)
```

## Technical tradeoffs
- **No bundler**: plain `tsc` keeps the project transparent for learners reviewing code.
- **Canvas for visuals**: lightweight, controllable, and framework-free.
- **Service worker**: minimal cache for offline replay; failures are non-fatal.

## Pedagogy notes
- Beads only appear where concrete part–whole intuition matters (fractions, ratios, probability).
- Beads intentionally **do not** represent irrationals; the gap illustrates endless decimals.
- Encouraging tone and calm palette reduce performance pressure—mastery over speed.
